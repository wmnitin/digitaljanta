Try this :-
  • Open RUN dialogue box using Windows + R button. Type Regedit and press ENTER. It will open the Registry Editor.
  • Now, Navigate to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control.
  • Now, right click on Control key and select New>>Key option. Name the new key as StorageDevicePolicies. If it already exists, then click on it
  • Now, come to the right pane of StorageDevicePolicies, right click here and select New>>DWORD Value. Name the new DWORD Value as WriteProtect. If the DWORD Value already exists then, double-click on it. Set the Value Data from 1 to 0. Click on OK button.
  • Close the Registry Editor and re-plug your USB or SD Card.
  • and try to Format.
