If you are using paid Internet and want to save data then, there is good news for you.
Go to
Setting -> Data Usage -> press menu and then check mark on Restrict background
It is very useful.
Helped me Save lots of data.

Disadvantage ->
- Notification from Apps like WhatsApp,Twitter, Facebook, Quora etc are not allowed until you disable it.
- play store requires background data

Advantage ->
- save lots of data
- enabling and disabling this function is very fast
- helpful when you are browsing (not downloading)
