GBWhatsApp – WhatsApp Mod [Dual WhatsApp in One Phone]

GBWhatsApp Mod is built on the latest version of the WhatsApp and allows you to run two WhatsApp at same time.

How to Install ?

  1. Take a backup of your chats in original whatsapp.
  2. Uninstall whatsapp.
  3. Install gbwhatsapp from link given below.
  4. Restore and use.
  5. Now install whatsapp from playstore and use another number to verify.
  6. Done, Enjoy 2 WhatsApps !

    Download Links :- https://uploadex.com/gf1p1elmhq9i OR https://dailyuploads.net/cx8ovyp6mz7i
