i had bought it too..
now figured out whats was problem
go to leoxsys LAN utility (from start menu)
>click MODE(after refresh option)(just where file,edit, etc menu are)
>now click acees point
now down below you will see option
>internet sharing option (or similar )
>do not choose auto option
>choose the LOCAL area connection(your broadband lan )

click apply
all done
now connect with mobile wifi
