How to make config file for any vpn.(Easiest Method Ever)
Requirements-
1. Internet connection.(for downloading config file)
2. A PC or Laptop(Not tried using Android phone)
3. Some knowledge about editing config file.

Steps-
1. Visit vpngate.net .
2. Find a good config file with high speed. (Japan and Korea are the fastest servers).
3. Download the config file in which the IP is inserted by default.
4. After downloading config file open it via Notepad.
5. Now press Ctrl+F and search for nobind and remove it from the config file.
6. Now under this line- Specify either 'proto tcp' or 'proto udp'. Write this line lport *** (***- It is the lport or rport).
7. Now save this and use the config file for unlimited usage.

Advantages-
1. No need of password file.
2. No need of registration or buying and premium account.
3. High speed config files.
