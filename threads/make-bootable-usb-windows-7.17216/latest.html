you just need 10 cmds to enter in cmd to make a bootable windows vista + after vista windows...as the setups contains bootmgr .
open cmd
commands:
1. diskpart
2. list disk
3. select disk {1,2,3,4 acc to the secondary memory you want to use}
/*be careful in entering this command check weather you have selected right disk or not*/
4. clean
5. create parition primay
6. select parition 1
7. active
8. format fs=fat32 quick
or
format fs=ntfs quick
/*ntfs preffered*/
9. assign
10. exit

Now just copy the extrated setup in you pen drive
