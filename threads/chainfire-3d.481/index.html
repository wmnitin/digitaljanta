Chainfire3D is an intermediary OpenGL driver. What does that mean? It means that Chainfire3D sits between your apps and the graphics drivers, and can intercept and/or change commands between the two. It has some built-in functions, and can be further extended with plugins to provide extra functionality. It is for low end devices. It will increase game performance and texture size and quality. If your game is running slowly and have texture problem then it will help you.
  • For Froyo and Gingerbread ONLY. NOT COMPATIBLE with Honeycomb/Ice Cream Sandwich/Jelly Bean !
  • ROOT is required for this program, as well as a writable /system partition. S-OFF is also required on HTC devices !!
  • Features include reducing texture size and quality to reduce memory use and potentially increase game performance, as well as increasing memory use potentially increase performance.
How to Use

  1. Extract plugins rar folder.
  2. Inside the folder, you`ll find libGLEMU_NVIDIA.zip, libGLEMU_POWERVR.zip and libGLEMU_QUALCOMM.zip (don`t extract those files), copy to your SDcard
  3. Open ChainFire3d application then install the ChainFire3D driver
  4. Reboot your device
  5. Open ChainFired3D then select option install plugins
  6. Automatically ChainFire3d will detect plugins files at sdcard root, install all of them.
  7. Back to ChainFire3D Menu then select Default open GL Settings.
  8. Choose Use Plugin and select NVIDIA/POWERVR/QUALCOMM.
  9. Minimize chainfire 3D and open game that you want to play.
  10. Done. Enjoy your 3d game.
