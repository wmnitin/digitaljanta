Vodafone all USSD codes *141# - Check main balance

*111*2# Recommended way to check main balance which also displays your mobile no

*444# - To activate GPRS /SMS plan(Not sure about this pl'z verify )

*142# - To know free sms balance

*145# - To know calling balance

Vodafone gprs data balance send GPRS to 144.

*111*5*2# - Check 3G data plan and remaining data balance.

To activate 3G :-- ACT3G to 111

*444*8# - Rs 8 - 20 MB 3G plan
