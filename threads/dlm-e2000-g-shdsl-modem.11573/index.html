~~~~~~~~~~~~~~~~~~~~~~~~~~~~~INTRODUCTION~~~~~~~~~~~~~~~~~~~~~~~~~

D-Link’s DLM-E2000 is a cost effective G.SHDSL modem transports multi-mega bit data stream over 2 wire using TC-PAM technology. This modem offers a choice of V.35 or 10x100M Ethernet or E1 interface or G.703 Co directional 64K interface or V.24/RS232 Sync/Async data interface.

SOME SPECIFICATION:-
•Meet ITU-T G.991.2 ETSI TS101 524.
•Supports DTE data rates from 64Kbps to 4624 Kbps per 64Kbps increment.
•Feature modular V.35, LAN, E1, G.703 64K Co-directional, RS232 user interface.
•Provides remote control and monitoring using in-band EOC channel.
•Supports DSL PRBS BER test.
•Supports V.54 local, remote and numbering loop back.
•Performs G.SHDSL and E1 line performance monitoring and statistics
•7*24 hours performance counters data storage.
•Detects and operates DSL loop crossover.
•Supports optional LCD display or SNMP management interface.
•Supports remote download and configuration.
•Supports optional receiving line feeding power from the remote end.
•DLM-E2000 supports multiple customer interfaces as LAN,1xFE1/E1,1xV.35, G.703,RS232 ,Interface module.
