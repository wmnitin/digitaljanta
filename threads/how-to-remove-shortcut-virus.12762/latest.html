Don't open that shortcut it's a Virus :/

Temporary solution to access your files in USB drive :-
  • Click on Tools tab.
  • Then Go to Folders option.
  • In View tab, hidden files and filders.
  • Tick Show hidden files and folders.
  • Untick Hide system files.
  • That's it ...!
  • Now you can access your files in usb drive.

type in :

2. attrib -h -r -s /s /d Drivename:\*.* (This command helps to unhide all hidden files and removes shortcuts)

3. If you are infected by malicious script , here is how to remove it.

Press Ctrl+Shift+Esc > opens Task Manager > go to process Tab, stop wscript.exe.

4. disable wscript.exe starting from boot , open Run>type in msconfig >go to startup tab disable wscript.exe.

5. Go to Run , type in regedit Navigate to HKEY_CURRENT_USER / Software / Microsoft / Windows / CurrentVersion / Run
