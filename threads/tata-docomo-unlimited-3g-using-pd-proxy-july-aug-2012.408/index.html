Download Pd proxy from official website. And also register there for username and password.

Now extract pd proxy and open it.

Goto advance settings and then proto option.

Enter Udp port 9201

Bind local port 53

Tcp port 443

and after saving them

goto main screen select demo server and udp

then enter username and password u got from registration. and then connect


Note first u have to connect to the net by using apn Tata.Docomo.internet.


you will get 100 mb daily in trial account.


To use more data make as many account data u wanted.

But this is most important u have to disconnect before completing 100 mb in one account ( Disconnect at 90 mb) and then connect through another account and use other 90 mb. keep this following as per ur need.

If u completed ur 100 mb in one account by mistake then u will not be able to use other account also in same pc. U will be able to use again after 9 p.m. indian time.

If u want hassel free unlimited net then buy premium account . Which very affordable Only 300 Rs for 1 month as compare to 3g prices in india.
