As u all. know that tez is not running in supersu rooted phone..
TO RUN TEZ IN ROOTED PHONE
1.U NEED TO DO FULL UNROOT BY USING SUPER SU SETTING
AND
also by flashing this unroot zip by custom recovery https://forum.xda-developers.com/showpost.php?p=63615067
2.If u need root than u have to flash magisk v14 or any version from custom recovery https://forum.xda-developers.com/apps/magisk/official-magisk-v7-universal-systemless-t3473445.(Note:google if ur phone support magisk or not)
3.open magisk then click on check safety net from it and if u get both ct profile and basic integrity true OR ct profile - false and basic integrity- true than u can run tez
4.if u also want xposed framework than u should use systemless xposed for ur phone for that u can Google for it and after flashing ur desired file u have to uncheck xposed from magisk module and reboot.. by unchecking u can run tez and if u want to use xposed than again check mark and use xposed module.

Note : tez will run only when safety net condition which i have given above appear and xposed is uncheck in magisk module

Before flashing magisk and systemless xposed u can google for this if ur device is supported or not i will be not responsible for any bootloop
