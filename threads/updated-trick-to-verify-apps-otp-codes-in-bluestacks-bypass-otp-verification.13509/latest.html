  2. BS-Tweaker - http://bit.ly/bluestack_tweaker (Need for Loots).
  3. Textra Messaging App
  4. Fake Text Message App
Steps to Verify OTPs In Bluestacks :-
  1. Firstly Open Bluestacks (Must be a Rooted Version - If not then make sure you download it from the above link).
  2. Install Textra Messaging App and Fake Text Message App (Both) in Bluestacks.
  3. Now Open Textra Message App and Click on "Make Default app" Option.
  4. Now if in case, i want to verify Ladooo app in Bluestack then -
    Firstly, I need to Install Ladoo App in Bluestack > Then I will an Unregistered Enter Mobile Number in Ladooo app and send OTP to Verify it.
  5. Now within seconds you will get an OTP Message to that number > NowSee from Which of the Name they sent OTP Message, Ladoo send Verify SMS from "MM-Ladooo" or"DM-Ladooo" etc.
  6. Move to Bluestacks again > Goto Contacts and Create a New Contact named same as you got messaged from like - "MM-Ladooo". Enter Mobile Number in which you received that message > Now Save it.

  7. Now Open Fake Text message app & Create new fake message from newly created Contact Enter Full Message Which you Got via SMS like in Ladoo message format is like this"ladooo verification OTP 792479" Set Time & Date Right After one minute & Click on"Fake" Button.
  8. So Now ! Right After 1 Minute Fake Messaging app will send same fake Message in Bluestacks and Ladooo will be accepting it as true one :p

Ladooo app successfully Verified the OTP .
So Now you have got to know what are the use of these two Important Apps - Textra Messaging App and Fake Text Message App .

  • Texta App will act as Default Messaging app in Bluestacks. Google Hangouts can also be used for this purpose also.
  • Fake Message app will send a fake message from the Same Number and Same format which you have applied.
