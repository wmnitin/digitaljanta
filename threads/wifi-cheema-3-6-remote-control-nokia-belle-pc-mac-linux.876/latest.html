WiFi Cheema is a free application that allows you to control your Windows, Mac or Linux based computer remotely. Watch and control your Desktop through your Nokia mobile. Ideal for Managers, students and Professionals for giving Presentations.

Features:

Simple GUI, offers an easy to use workflow and highly polished visual cues.
Servers are Compatible with Microsoft Windows, Mac OS X and Red Hat Linux Clients available for Nokia,Samsung and other J2ME based mobilephones
Use your smartphone as a Trackpad or mouse View your Desktop remotely on your mobilephone Secure Verisign Signed Server and Client Software to ensure security Ideal for giving Microsoft PowerPoint or Apple Keynote presentations


How to use?

Install WiFiCheema CLIENT on your mobile phone Install WiFiCheema SERVER for your computer (PC / Mac / Linux) from the download package below.

Install WiFiCheema SERVER on your computer.
ALLOW INCOMING CONNECTIONS if prompted Start WiFiCheema Server and note down the IP Address.

Start WiFiCheema Client on your phone and press CONNECT button that appears after the Ad screen Enter the IP Address for WiFiCheema SERVER and press CONNECT Your computer screen will appear on your mobile.

Enjoy
