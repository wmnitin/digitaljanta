How to Unlock Bsnl 3G data card (Teracom LW272&273)​
  • INSTALL THE SOFTWARE ====Here
  • INSERT OTHER SIM CARD EX:- DOCOMO & FOLLOWS THIS STEPS
  1. Install the USB modem as usual
  2. Create a new APN in the BSNL interface (for docomo it is "tata.docomo.internet").
    Make it as default.
  3. Check in Windows under installed modem options, there you will have HSPA DataCardPropritary USB MODEM
  4. Create a new Dialup with your desired name and use the above modem as modem, dialing number will be *99#, user name and password will be blank
  5. Now remove your datacard and connect it again to the system.
  6. Now remove your datacard and connect it again to the system.
  7. The green light will glow after 10-15 seconds. Now you can connect using the new dialup connection we made. (Here onwards, every time you can use the dial-up)
It will get connected in 3.6 MBPS. actual speed will be depended on the SIM you are using and the network strength.
