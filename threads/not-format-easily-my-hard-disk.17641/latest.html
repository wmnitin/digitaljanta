1. Re-start your PC and then start pressing (F8) key then to open Advanced Boot Options screen
2. Select Safe Mode with Command Prompt
3. Find the drive letter , (G) for example then write as
4. C:\windows\system32>G: and press enter
5. G:\>format G: and press enter
6. If ask you (Y/N) then press (Y) and press enter
7. This should do the trick.
