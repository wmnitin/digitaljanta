I had the same issue with MMX352G.....but I have already sorted it far earlier.

What you have to do....
  1. Uninstall Everything as It was earlier to begin from starting.
  2. Restart you PC
  3. Plug your Modem and install Software. Now it will show you message Driver installation faily. But GUI will be installed perfectly.
  4. (IMPORTANT) Go to C:\Program Files\"Modem Name"\Driver
  5. Execute "InstallDriver.exe" as Administrator. Wait... Now it will Show you Driver Installation Fail. (Don't Worry)
  6. Now go to Device manager.
  7. In Device Manager you will see 4 or 3 drivers under "Other Devices" Option having Same name.
  8. Right Click on First one and Update Driver with Custom Path :-
64bit :- C:\Program Files\"Your Modem"\Driver\Win7_64
32bit :- C:\Program Files\"Your Modem"\Driver\Win7_32

Now It should be install Correctly.

and Repeat the same for Remaining. Drivers

Hope It will Help you.

NOTE : Follow the Step in Sequence.
