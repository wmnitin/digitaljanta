What is a blog?
Blog has been derived from the word WEB-LOG . Means
the log created by web(i.e, the event and log files of any
site.)
Blog is a Tool to represent the activities
any website.
But you really don't need to have a
website to create a blog.
The blog can be created with two types of packages .
A . A free blog- blogger.com (have some
limitations)
B. A paid blog- WordPress.org (can come through
limitations)
