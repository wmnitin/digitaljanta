Description -:

*MicroZip is a powerful file compression and encryption utility for mobile phones which allows you to create and extract compressed archives in multiple formats and encrypt sensitive information with powerful AES-256 encryption. MicroZip can create and extract ZIP,TAR,TGZ,TBZ2 archives and extract
many other formats as JAR,GZ,BZ2,TBZ,WAR,BZIP2,GZIP,Z,TAZ,CBZ etc.

*MicroZip allows you to extract selected files without extracting the whole ZIP archive.
MicroZip also supports encrypted ZIP archives with both classic ZIP encryption and AES 256 encryption algorithm.

Features-:
  • Support for multiple archive formats (ZIP,TAR,TGZ, TBZ2)
  • Encrypt sensitive data with powerful AES-256 encryption. (WinZipAES-256)
  • Extract password protected Zip files with ZipCrypto and AES-256 algorithm
  • Simple and stylish user interface
  • Capable of extracting single file without extracting whole zip file
  • High portability. Supports any j2me device with MIDP 2.0 and CLDC 1.0
  • Low memory footprint
  • Extract only support for various archives: JAR,WAR,CBZ,Z,TAZ etc!
