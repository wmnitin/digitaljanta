Many user are facing issue as they are not getting speed while download.

1. For browsing jio gives 60-70ms ping so use VPN like hotspot shield or hola vpn or other you may like.
2. For downloading,
i) DO NOT USE ANY VPN
ii) keep 10-15 torrent files together in utorrent and set active number of torrent to all and wait for 15 mins to see results (this will give 4-5 mbps constant in aggregate download)
iii) always check speed on google play store and youtube if they are giving speed then your card and tower and etc are perfect its just you. !
iv) IMP, go to settings - select network selection to manual - select jio 4g or jio IND , then go to preferred network and select 3G then again 4G this will re connect your jio sim to network or turn airplane mode on and off.
v) Set apn to

Name: anything you like
APN: jionet
Proxy: NOT SET
Port: NOT SET
Username: NOT SET
Password: NOT SET
Server: www.google.com or NOT SET
MMSC: NOT SET
MMS Proxy: NOT SET
MCC: 405
MNC: 857 or 863 or 874
Auten type: NOT SET
APN type: NOT SET
APN Protocol: IPv4/IPv6
APN Protocol Roaming: IPv4/IPv6
APN ENABLE
Bearer: Unspecified
MVNO Type: none.
MVNO value: not set

This are default setting no tweak are done.

So hope you guys try and get speed nearby 4-5mbps in torrent and zbigz.
