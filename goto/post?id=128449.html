www.4shared.com/get/DEHu4aDt/Uc_Browser_86_Handler.html

● Install & Open it.

● Create a setting/APN setting in your device as per mentioned below→

● APN: www

●Proxy: 208.116.58.26

●Port: 80

Save the setting. .

Now Open the handler and fill the text boxes as below→

● Proxy Type: http

●Proxy server: omi.vodafone.com
