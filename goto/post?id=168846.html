1. First of all install myjio app v3.2.05 and then install apk editor from play store.

2. Then Open apk editor then select apk from app option

3 after that click on myjio and slect option common edit

4. Then Change the Version code - 3320 and version name 3.2.20 and click save .

5. then minimise apk editor and unistall myjio

6. Open apk editor and install modified apk.

and we are done.Thankyou​
