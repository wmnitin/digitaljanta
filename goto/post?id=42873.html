How To Use Airtel's FreeZone Google Services on PC Mozilla Firefox users need to add User Agent Switcher extension on their browser , you can add this extension by navigating this link. https://addons.mozilla.org/en-US/firefox/addon/user-agent-switcher/ Afer that Goto Tools >> User Agent Switcher >> Edit User Agents Add New User Agent , Put OperaMini in Description and in the User Agent box put the following user agent string. Opera/9.80 (J2ME/MIDP; Opera Mini/5.0.0.351 Profile/22.478; U; en) Presto/2.5.25 Version/10.54 Click on ok to save the user agent now select the newly created operamini user agent that all . Now open the following link on your browser. airtel.in/freezone
