1. First of all download ccleaner and clean your pc and registry.
2. Run Disk cleanup (Control panel -> system & security -> administrative tools -> disk cleanup) (also clean system files)
3. Go to Task Manager -> Startup and disable software which are unnecessary at start-up.
4. Remove all program which you don't need.
5. Go to system protection and delete your system restore cache. (Right click on my computer -> properties -> system protection -> configure -> delete)
