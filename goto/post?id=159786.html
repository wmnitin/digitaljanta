Trick To Use Jio Sim In All Phones.

Didn't Tried It Personally, Try & Comment If It Works For You.

1. FIRST OF ALL GET UR SIM READY. INSERT THAT TO SLOT NO.1.

2. TAKE A BACKUP OF ALL UR DATA IN INTERNAL STORAGE.

3. FLASH OFFICIAL COS-12.1 (its work only one plus )THROUGH FASTBOOT ON UR DEVICE.

4. AFTER BOOTING TO COS. SKIP THE SETUP-WIZARD STEPS.

5. GO TO SETTIINGS. FIRSTLY CREATE AN APN

APN-NAME-INTERNET
APN-JIONET
PROXY-NOT SET
SKIP PORT ,USERNAME, PASSWORD, SERVER, MMSC, MMS PROXY,MMS PORT,MCC,MNC TO DEFAULT VALUES.

THEN, AUTHENTICATION TYPE - NONE
APN TYPE - DEFAULT,IA

SAVE THE APN AND MAKE IT DEFAULT.

NETWORK SHOULD COME IN 10-20 SECONDS.

IF NOT,

SET LTE ONLY MODE FROM *#*#4636#*#* - PHONE INFORMATION TRICK.

ONCE NET GOT CONNECTED AND WORKING IN COS-12.1 . U CAN NOW MOVE TO ANY OTHER ROM. ANY ROM OF UR CHOICE.

JUST REMEMBER. AFTER SETTING UP IF U USE ANY ROM.
SET APN (MOST PRECIOUS STEP)
SET LTE MODE. AND NET WILL START WORKING.

AND FOR THOSE WHO WANT VOLTE TO WORK. DOWNLOAD JOINJIO APP FROM GOOGLE PLAY. IT WILL SIGN U IN FROM UR SIM CARD AND U CAN CALL FROM THAT APP (HD CALLS).


Use Own Risk....
