HELLO DJ

HERE ARE SOME SPECIAL CODES BY WHICH YOU CAN CHANGE WHATSAPP FONT OR INSERT SPECIAL TEXT....


1. MAKE CHARACTERS BOLD

PUT * AT THE START AND END OF THE TEXT WHICH YOU WANT TO MAKE BOLD

EX:- *HELLO* --->>>> HELLO



2. MAKE CHARACTERS ITALIC

PUT _ AT THE START AND END OF THE TEXT WHICH YOU WANT TO MAKE ITALIC

EX:- _HELLO_ --->>>> HELLO



3. MAKE CHARACTERS STRIKE DOWN

PUT ~ AT THE START AND END OF THE TEXT WHICH YOU WANT TO STRIKE DOWN

EX:- ~HELLO~ --->>>> HELLO




4. MAKE A COMBINATION OF THE ABOVE 3

EX:- *_HELLO_* --->>>> HELLO

EX:- *_~HELLO~_* --->>>> HELLO
