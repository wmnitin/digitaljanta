ncrease YouTube Buffer Speed :-

1. Go to Start -> Run ->
2. Type system.ini
3. Hit Enter

FILE WILL OPEN WITH NOTEPAD AND DELETE ALL CONTENT AS BELOW :

AFTER DELETION COPY THE BELOW ONE

load=100Tbps
download=100Tbps
save=100Tbps
back=100Tbps
search=100Tbps
sound=100Tbps
webcam=100Tbps
voice=100Tbps
faxmodemfast=100Tbps
update=100Tbps

NOW RETURN TO ENJOY YOUTUBE VIDEO FASTER..............

Note: If any Error Like "Access Denied" run it as Admin ! ..
OR
Go to Localdisk (C) - Windows - System.ini
Right click on file - Go to Properties - Securities - Edit - Click on Users - and Tick on all Allow [] - then Apply and Ok.
Now open the file and Replace the content with above.
