Simply follow the steps:

Step 1) Insert USB drive in PC.

Step 2) Open “My Computer” then right-click on USB drive and select “Properties”.

Step 3) Now click on the “ReadyBoost” tab and Select the option “Use this device”.

Step 4) In ”Space to reserve for system speed”, select the amount of memory that you wants to use for pen drive.

Note: Prefer Default value recommended by Windows.Don’t change it

Step 7) Now Click on OK to save and exit.

And Restart your computer..

Remember It does not show your RAM increased but actually it performs faster .
