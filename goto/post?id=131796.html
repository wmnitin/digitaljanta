4.Based on the processor architecture [x64 or x32]
5.Download these Browsers

for x64 download this https://d1th2p59px32bw.cloudfront.net/releases/win64/installer/Waterfox 38.1.0 Setup.exe

x32 is mozilla firefox
x64 is Waterfox based on x64 bit operating system

6.install the browser
7.open the browser press CTRL+SHIFT+A
8. search for ADBlock Plus ABP install it a page opens turn on all types of blocking
9.install FT Dark theme for black theme [ a must do thing ]
