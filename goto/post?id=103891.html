For non premium users: Download resume same file from different host if the current host don't support resume

Even some hosts doesn't support resume in Jdownloader, So after partially downloaded if its interrupted and you tried to resume, then jdownloader will start the file to download from beginning after long waiting time from hosts like luckyshare.com, terafile.co etc.

Only Show Download URL option is available in Jdownloader but you can't change link.
So, to change the download URL and start resume the download from different host 1st goto
Jdownloader installed folder.
Mine is

C:\Users\Lenovo\AppData\Local\JDownloader 2.0\ (change to yours)

Open "cfg" folder

Now look for a zip file starts with download.... example "downloadList5900.zip" in some case there may be more than 1 such file.

Inside the zip there are file with name 00, 01 , 02, 03, 04 ...........etc

View all the files till you find the download URL

Now , example if your Download Link is in 06 file then extract it any where in your PC and drag it in Notepad and change the url
Example

http://luckyshare.net/1234/xyz.rar

to

http://turbobit.net/xyz.html

also chnage the host from

http://luckyshare.net
to
http://turbobit.net



Now save and replace with old in zip

Open jdownloader and click Force Download Starts .The download will resume from new host.
