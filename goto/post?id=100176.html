It's main purpose is not to increase your downloading speed directly.
It will eliminate windows bottleneck so u can get max. speed that u can possibly get on your network.

Follow these steps:
1.open command prompt run as admin.
2.(type) netsh int tcp show global
Click enter
3.int tcp set global chimney=enabled
Click enter
4.netsh int tcp set global autotuninglevel=normal
Click enter
5.netsh int tcp set global congestionprovider=ctcp
Click enter
6.netsh winsock reset
Click enter


Restart PC......

Open cmd as admin again......
Type
1.Ipconfig
Click enter
2.ipconfig /flushDNS
Click enter

Restart PC.......
