Requirements : Any LINUX LIVE CD only , for simplicity i Suggest PUPPY LINUX. [LIGHTEST LINUX of about 200-230Mb Only]

Steps :
- Insert Live CD and BOOT from it. You may Make it Bootable either by USB Flash by using this.
[ You may need to turn OFF Security BOOT Features for UEFI Bios ]
- Locate the drive where WINDOWS is installed.
[ Goto Windows/System 32 Folder ]

[ Let it be "cmd0.exe" ]
- Search another File with name "sethc.exe" and rename it to "cmd.exe".
- Finally rename "cmd0.exe" to "sethc.exe".
- SHUTDOWN and reboot your System.
- Press Shift Key 5 - 6 Times a the login screen.
[ This'll open the Command Prompt ]
- Type "net user" without Quotes and hit ENTER.
[ This'll shows you the list of Active Users Accounts ]
- Type " net user<space><Your User Account Name><Space>* ".
[ This'll display you to enter New Password ]
- Enter any New Password and Confirm or leave it blank to remove password.
