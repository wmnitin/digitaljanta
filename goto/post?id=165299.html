firstly
never buy anything (i mean anything) on ebay with a seller feedback negative or not at all(0 feedback)..
it's the thumb rule of ebay
secondly
you can't cancel the item as it's against ebay policy and only the seller from his/her side can cancel by mutual understanding..
lastly things you can do-(your money is with ebay currently but it can return to you)
1. notify ebay
2. try contacting the seller and ask him to cancel
3. when the package arrives don't claim it and let it return
4. accept item and then don't open the package and then return asking ebay

there is 50-50 chance for your money to be safe..
check for similar offers on ebay with (a different seller but with again 0 feedback) might be he the same guy..
