Acer launches Intel powered Iconia W3, world’s first 8.1-inch Windows 8 tablet:-



Taiwanese firm Acer has launched world's first 8.1-inch Windows 8 tablet the Iconia W3 is which is believed to be the smallest tablet running on the latest operating system from Microsoft.
The company is offering the device for Rs 27,999 till end of July. After that it will be priced at Rs 30,499.
Iconia W3 has the 8.1-inch display with 1280×800 pixels resolution. Powered by Intel Atom chipset backed by 2GB RAM and 32GB memory Iconia W3 comes with 2-megapixel front and rear cameras.
Iconia W3 comes with micro-HDMI and micro-USB ports and Acer and 6800mAH battery which the company claims provides up to 8 hours usage. Acer is offering an optional full-size Bluetooth keyboard holds the tablet upright for typing. The tablet docks to the keyboard's rear for mobility. The keyboard, similar to those on 13.3-inch notebooks, is comfortable to use.
